## Install
- Homepage and documentation
  - [LiuYing](https://tix007.github.io/vue3-liuying-ui/)

## Install
```shell
npm install liuying-ui -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import LiuYing from 'liuying-ui'

Vue.use(LiuYing)

// or
import {
  Select,
  Button
  // ...
} from 'liuying-ui'

Vue.component(Select.name, Select)
Vue.component(Button.name, Button)
```
